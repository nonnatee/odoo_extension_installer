Odoo Extension Installer Documentation
======================================

Welcome to the documentation for the Odoo Extension Installer module (targeting Odoo 19.0). 
This tool provides a secure interface for Odoo settings administrators to install open-source third-party Odoo modules via GitHub or ZIP file uploads.
